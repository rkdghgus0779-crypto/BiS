# Code of Conduct

BiS is a learning-focused open-source project. Everyone participating in this project is expected to help keep the space respectful, patient, and welcoming.

## Expected Behavior

Participants should:

* Use welcoming and inclusive language
* Respect different backgrounds and experience levels
* Give constructive feedback
* Focus on what is best for learners and the project
* Be patient with beginner questions

## Unacceptable Behavior

Unacceptable behavior includes harassment, insults, personal attacks, discriminatory language, repeated disruption, or sharing private information without permission.

## Reporting

If you see behavior that does not fit this code of conduct, please open a private conversation with the maintainer through the contact information on the maintainer's GitHub profile.

Reports will be reviewed with care. Maintainers may remove comments, close issues, or block participants when needed to protect the community.
