# Contributing to BiS

Thank you for considering a contribution to BiS. This project is intentionally beginner-friendly, so documentation improvements and small examples are just as valuable as code.

## Good First Contributions

Useful contributions include:

* Fixing unclear explanations in the README or docs
* Adding short biology, neuroscience, or bioinformatics example notes
* Adding tests for existing utilities
* Improving CLI error messages
* Suggesting public datasets that are friendly for beginners

## Local Setup

```bash
git clone https://github.com/rkdghgus0779-crypto/BiS.git
cd BiS
python -m pip install -e .
python -m unittest discover -s tests
```

## Pull Request Checklist

Before opening a pull request, please check that:

* The change has a clear purpose
* Tests pass locally with `python -m unittest discover -s tests`
* New examples are readable for beginners
* Documentation is updated when behavior changes

## Project Values

BiS values readable code, reproducible examples, and patient explanations. A small clear contribution is better than a large change that is hard for students to understand.
