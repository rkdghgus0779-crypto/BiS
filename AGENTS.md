# AGENTS.md

BiS is an early-stage open-source educational project for biology, bioengineering, neuroscience, bioinformatics, and computational biology students.

## Rules for AI Coding Agents

- Keep claims honest. Do not invent users, downloads, citations, contributors, or adoption.
- Prefer beginner-friendly explanations.
- When adding biology content, explain the biological meaning, not only the code.
- Keep changes small and reviewable.
- Update tests when Python behavior changes.
- Update documentation when examples or CLI behavior changes.
- Use clear Markdown structure.
- Avoid medical advice or clinical claims.

## Setup and Test Commands

```bash
python -m pip install -e .
python -m unittest discover -s tests
```
