# Impact and Reach

BiS is currently an early-stage open-source project. Its first goal is to make the bridge between biology concepts and Python-based computational thinking easier for students.

The project's direction is biomedical AI education: helping learners move from biological questions to reproducible code, documentation, and analysis workflows.

## Current Audience

The project is designed for:

* Students in bio and brain engineering
* Beginners learning bioinformatics or computational biology
* Learners who want simple, readable examples before using larger scientific Python libraries

## Current Project Evidence

Current public evidence includes:

* MIT-licensed public repository
* Beginner-friendly README and example note
* Python package structure
* Command line interface
* Unit tests and CI workflow
* Contribution, conduct, and security documentation
* Research focus documentation around bioinformatics, single-cell thinking, fibrosis, immune response, and biomedical AI

## Impact Goals

Near-term goals:

* Publish the first tagged release
* Add public dataset examples
* Add beginner issues for new contributors
* Document real student use cases
* Track stars, forks, contributors, merged pull requests, and package downloads
* Contribute documentation and examples to established scientific Python projects

Longer-term goals:

* Become a small educational toolkit for students entering bioinformatics and computational biology
* Provide reproducible examples that connect biological concepts to code
* Create a friendly first-contribution path for students interested in scientific open source
