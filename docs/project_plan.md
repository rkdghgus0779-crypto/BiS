# Project Plan

## Goal

The goal of BiS is to build a beginner-friendly open-source resource for students interested in bio and brain engineering, bioinformatics, neuroscience, and computational biology.

## Current Stage

This project is currently in its early stage. Version 0.1 focuses on documentation, a small Python package, a command line interface, tests, and basic learning materials.

## Planned Improvements

- Add more biology-related example notes
- Create simple Python scripts for organizing study materials
- Add examples for biological data analysis
- Write beginner-friendly explanations of computational biology concepts
- Explore AI-assisted workflows for scientific learning
- Add GitHub issues labeled `good first issue`
- Publish tagged releases and track project usage
- Add tutorials that use public datasets
- Add a research focus track for biomedical AI, single-cell thinking, fibrosis, and immune response

## Open Source Growth Plan

To grow BiS into a stronger open-source project, the maintainer will focus on:

- Making setup and contribution paths easy for first-time contributors
- Keeping all examples reproducible with public data
- Tracking stars, forks, issues, pull requests, releases, and downloads
- Writing contributor-friendly issues with small scopes
- Contributing related documentation and examples to external scientific Python projects
- Keeping personal interests organized under one project theme: biomedical problems explained with data and code

## Long-Term Vision

BiS aims to become a public learning archive and small educational toolkit for students who want to connect biological knowledge with coding and data-driven scientific thinking.
