# v0.1.0 - Early BiS learning toolkit

This is the first early release of BiS as a beginner-friendly open-source learning toolkit for bio and brain engineering, bioinformatics, computational biology, and AI-assisted scientific learning.

## Included

- Python package structure
- `bis-notes` command line interface
- Simple note summarization
- Keyword extraction
- JSON output support
- Beginner documentation
- Bioinformatics glossary
- Single-cell RNA-seq starter reading guide
- Fibrosis and immune response case-study outline
- CI workflow and unit tests
- `AGENTS.md` guidance for AI coding agents

## Current status

BiS is still an early-stage educational project. The next goal is to add public dataset tutorials, single-cell RNA-seq beginner materials, and more biomedical learning examples.
