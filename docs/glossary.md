# Beginner Bioinformatics Glossary

This glossary defines common beginner terms for bioinformatics, computational biology, and biomedical AI learning.

BiS is an educational project. These definitions are not medical advice.

## Terms

| Term | Beginner explanation |
| --- | --- |
| Bioinformatics | Using computation to store, analyze, and interpret biological data. |
| Computational biology | Using computation, statistics, and models to understand biological systems. |
| Gene expression | A measure of how active a gene is in a cell or sample. |
| RNA-seq | A sequencing method used to measure RNA abundance. |
| Bulk RNA-seq | RNA-seq from a mixture of many cells, producing an average signal. |
| Single-cell RNA-seq | RNA-seq that measures gene expression in individual cells. |
| Count matrix | A table of expression counts, often genes by cells or genes by samples. |
| Metadata | Information describing samples or cells, such as condition, tissue, batch, or cell type. |
| AnnData | A common Python data object for storing single-cell matrices, metadata, and results. |
| `.h5ad` | A common file format for saving AnnData single-cell datasets. |
| Clustering | Grouping cells or samples with similar patterns. |
| Batch effect | Technical variation caused by processing differences rather than biology. |
| Marker gene | A gene used as a clue for identifying a cell type or cell state. |
| Differential expression | Comparing gene expression between groups to find changed genes. |
| Fibrosis | Excessive tissue repair that can cause extracellular matrix accumulation and stiffness. |
| Fibroblast | A stromal cell that produces and remodels extracellular matrix. |
| Myofibroblast | An activated fibroblast-like state often linked to matrix production and contraction. |
| Macrophage | An immune cell involved in inflammation, cleanup, repair, and cell signaling. |
| Extracellular matrix | The structural network around cells; it can accumulate abnormally in fibrosis. |
| TGF-beta | A signaling molecule often discussed in fibroblast activation and fibrosis. |
